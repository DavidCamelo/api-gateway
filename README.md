## API Gateway - [Open API](https://spring-boot.davidcamelo.com/swagger-ui/index.html)
[![Infra Deployment](https://github.com/DavidCamelo/api-gateway/actions/workflows/deploy.yml/badge.svg)](https://github.com/DavidCamelo/api-gateway/actions/workflows/deploy.yml)

This project is based on Spring Boot and serves as an API Gateway for routing requests to various microservices within the application ecosystem. It leverages Spring Cloud Gateway for dynamic routing, load balancing, and security features.
